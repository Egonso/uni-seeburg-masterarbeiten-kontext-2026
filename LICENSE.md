# Lizenzhinweis

Dieses öffentliche Repository enthält kuratierte Betreuungsnotizen, Quellenverweise und Zusammenfassungen.

Soweit nicht anders angegeben, wird keine Open-Source-Lizenz eingeräumt. Inhalte dürfen gelesen und verlinkt werden. Eine Weiterverwendung, Veröffentlichung oder kommerzielle Nutzung braucht eine gesonderte Freigabe der jeweiligen Rechteinhaber.

Externe Quellen bleiben Eigentum der jeweiligen Rechteinhaber. Dieses Repository verlinkt offizielle Dokumente und Referenzseiten, statt vollständige Fremdtexte zu republizieren.
