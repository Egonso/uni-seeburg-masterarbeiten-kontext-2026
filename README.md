## Was bewusst nicht enthalten ist

Dieses Repository ist öffentlich. Deshalb enthält es keine privaten oder personenbezogenen Rohdaten:

- keine WhatsApp-Verläufe, Sprachnachrichten, Audios oder Chat-Medien,
- keine Mailkopien, E-Mail-Adressen aus Verläufen oder Message-IDs,
- keine vorbefüllten Formulare, Geburtstage, Adressen, Telefonnummern oder Signaturen,
- keine Modul-5-Bewertungen, Punkte, Matrikelnummern oder internen Feedbackauszüge,
- keine Pitch-Videos, Originalabgaben, ZIPs oder Rohtranskripte,
- keine individuellen Arbeitstitel, Forschungsthemen oder Themenstände der drei Arbeiten,
- keine vollständigen Kopien fremder PDF-/DOCX-Quellen, wenn Links und Zusammenfassungen reichen.

Wenn ein Thema personenbezogene Daten, Unternehmensdaten, Interviewdaten oder nicht freigegebene Unterlagen enthält, gehört es nicht in dieses Public-Repo.
